制作物について

出典・参考：

  • アニメーションロジック：以下のMDN Web Docsリファレンスを参考に3D回転動作を実装。
    • MDN: transform - CSS
    • MDN: perspective - CSS
    • MDN: backface-visibility - CSS

実装の工夫点：

  • データ管理の効率化： JavaScriptの配列を用いて単語データを管理。将来的な単語の追加や変更が容易な設計にしています。
  • 視覚的フィードバック： perspectiveとrotateYを組み合わせ、物理的なカードを模した直感的なUIを構築しました。
  • 学習支援機能： 基本的な表示機能に加え、記憶の定着を助けるための「ランダムシャッフル機能」をJavaScriptで独自に実装しました。
